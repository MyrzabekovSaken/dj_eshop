## General info
This project about online shopping daily products.
	
## Technologies
Project is created with:
* Python
* Django REST Framework
* HTML/CSS/JS
* AJAX
* Bootstrap
* Razorpay
	
## Setup
To run this project, install dependencies using poetry:

```
poetry install
```

Run project
```
python manage.py runserver
```
